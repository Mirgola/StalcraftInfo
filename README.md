Программа рассчитывает необходимое количество ресурсов для выбранных предметов бартера.

ВАЖНО:  
1. Программа требует минимальное разрешение 1540*900
2. Java SE Development Kit 21.0.8 и выше
https://www.oracle.com/java/technologies/javase/jdk21-archive-downloads.html
3. Запускать программу через файл StalcraftInfo.jar
